This document provides additional resources, books, articles, and courses to deepen your understanding of security and DevOps concepts covered in Chapter 9, with the most up-to-date resources for 2025.

## Books

### DevOps and CI/CD

1. **"The DevOps Handbook: Second Edition"** by Gene Kim, Jez Humble, Patrick Debois, John Willis, and Nicole Forsgren (2023)
   - [https://itrevolution.com/product/the-devops-handbook-second-edition/](https://itrevolution.com/product/the-devops-handbook-second-edition/)
   - Updated comprehensive guide to DevOps principles and practices with new case studies

2. **"DevOps Paradox: The Path to Greater Agility, Stability, and Security" (2025 Edition)** by Viktor Farcic and Darin Pope
   - [https://www.amazon.com/DevOps-Paradox-Greater-Stability-Security/dp/1839648473/](https://www.amazon.com/DevOps-Paradox-Greater-Stability-Security/dp/1839648473/)
   - Explores the tensions between agility, stability, and security in modern DevOps

3. **"Modern Software Engineering"** by David Farley (2024)
   - [https://www.pearson.com/en-us/subject-catalog/p/modern-software-engineering/P200000000545](https://www.pearson.com/en-us/subject-catalog/p/modern-software-engineering/P200000000545)
   - From the co-author of "Continuous Delivery" - focuses on software engineering fundamentals

4. **"Infrastructure as Code: Dynamic Systems for the Cloud Age (2nd Edition)"** by Kief Morris (2024)
   - [https://www.oreilly.com/library/view/infrastructure-as-code/9781098114671/](https://www.oreilly.com/library/view/infrastructure-as-code/9781098114671/)
   - Updated thorough coverage of modern infrastructure automation

### Kubernetes and Cloud Native

1. **"The Kubernetes Book: 2025 Edition"** by Nigel Poulton
   - [https://www.amazon.com/Kubernetes-Book-Nigel-Poulton/dp/1916585205/](https://www.amazon.com/Kubernetes-Book-Nigel-Poulton/dp/1916585205/)
   - Latest edition of the popular guide to Kubernetes, completely updated for 2025

2. **"Kubernetes: Up & Running (3rd Edition)"** by Brendan Burns, Joe Beda, Kelsey Hightower, and Lachlan Evenson
   - [https://www.oreilly.com/library/view/kubernetes-up-and/9781098110192/](https://www.oreilly.com/library/view/kubernetes-up-and/9781098110192/)
   - Latest edition of this practical guide to Kubernetes fundamentals

3. **"Cloud Native DevOps with Kubernetes (2nd Edition)"** by John Arundel and Justin Domingus
   - [https://www.oreilly.com/library/view/cloud-native-devops/9781098116293/](https://www.oreilly.com/library/view/cloud-native-devops/9781098116293/)
   - Updated guide to DevOps practices for Kubernetes

4. **"Platform Engineering on Kubernetes"** by Mauricio Salatino and Dinis Cruz (2024)
   - [https://www.manning.com/books/platform-engineering-on-kubernetes](https://www.manning.com/books/platform-engineering-on-kubernetes)
   - Modern approaches to building developer platforms with Kubernetes

### Security

1. **"DevSecOps: A Leader's Guide to Producing Secure Software"** by Larry Maccherone (2024)
   - [https://www.amazon.com/DevSecOps-Leaders-Producing-Secure-Software/dp/1119901057/](https://www.amazon.com/DevSecOps-Leaders-Producing-Secure-Software/dp/1119901057/)
   - Leadership perspective on integrating security into DevOps

2. **"Cloud Native Security"** by Liz Rice (2024)
   - [https://www.oreilly.com/library/view/cloud-native-security/9781492096627/](https://www.oreilly.com/library/view/cloud-native-security/9781492096627/)
   - Comprehensive guide to securing cloud native applications

3. **"Securing the Software Supply Chain"** by Michelle Dufty and Andres Vega (2025)
   - [https://www.packtpub.com/product/securing-the-software-supply-chain/9781805127062](https://www.packtpub.com/product/securing-the-software-supply-chain/9781805127062)
   - Focusing on securing the entire software delivery pipeline

4. **"Implementing Zero Trust Security"** by Jason Garbis and Jerry Chapman (2024)
   - [https://www.wiley.com/en-us/Implementing+Zero+Trust+Security-p-9781119884859](https://www.wiley.com/en-us/Implementing+Zero+Trust+Security-p-9781119884859)
   - Practical guide to implementing zero trust security models

### Site Reliability Engineering

1. **"Implementing Service Level Objectives"** by Alex Hidalgo (2024)
   - [https://www.oreilly.com/library/view/implementing-service-level/9781492076803/](https://www.oreilly.com/library/view/implementing-service-level/9781492076803/)
   - Comprehensive guide to SLOs, SLAs, and error budgets

2. **"Practical MLOps and SRE"** by Hwee Kai Tee (2025)
   - [https://www.packtpub.com/product/practical-mlops-and-sre/9781805128434](https://www.packtpub.com/product/practical-mlops-and-sre/9781805128434)
   - Combining ML operations with site reliability engineering

3. **"Chaos Engineering: Reliable Systems from Unreliable Components (2nd Edition)"** by Nora Jones and Casey Rosenthal (2024)
   - [https://www.oreilly.com/library/view/chaos-engineering-2nd/9781492057314/](https://www.oreilly.com/library/view/chaos-engineering-2nd/9781492057314/)
   - Updated guide to chaos engineering principles and practices

4. **"Observability Engineering"** by Charity Majors, Liz Fong-Jones, and George Miranda (2024)
   - [https://www.oreilly.com/library/view/observability-engineering/9781492076438/](https://www.oreilly.com/library/view/observability-engineering/9781492076438/)
   - Comprehensive guide to observability in modern systems
